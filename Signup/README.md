# BUW Study Matching Platform

## 📌 Description du projet

BUW Study Matching Platform est une plateforme web destinée aux étudiants de l’Université de Wuppertal permettant de trouver facilement des partenaires de révision et de créer des connexions académiques selon les cours, facultés et semestres.

L’objectif principal du projet est de simplifier la recherche de partenaires d’étude et de favoriser la collaboration entre étudiants.

---

# 🎯 Objectifs du projet

* Permettre aux étudiants de créer un profil universitaire
* Faciliter le matching entre étudiants ayant les mêmes cours
* Améliorer l’organisation des groupes de travail
* Offrir une interface moderne et intuitive
* Développer un projet frontend moderne en HTML, CSS et JavaScript

# ✨ Fonctionnalités actuellement implémentées

## ✅ Interface utilisateur moderne

* Design responsive
* Layout en Flexbox
* Cartes modernes avec ombres et animations
* Interface inspirée des plateformes modernes

## ✅ Système d’inscription

Les utilisateurs peuvent :

* Ajouter leur nom complet
* Ajouter leur email universitaire
* Ajouter leur faculté
* Sélectionner leur semestre
* Ajouter une description personnelle
* Ajouter une photo de profil (optionnelle)

---

## ✅ Validation en temps réel

Le formulaire vérifie automatiquement :

* La validité de l’email universitaire
* Les champs obligatoires
* La complétion du profil

Les validations sont affichées dynamiquement avec :

* ✅ Champs valides

* ➖ Champs optionnels
